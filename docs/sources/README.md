# Sources

Authoritative references used to verify historical data in itihas.

Each file corresponds to a data module and lists the primary sources consulted
for dates, classifications, and factual claims. General references that span
multiple modules are in [general.md](general.md).

| File | Module | Coverage |
|------|--------|----------|
| [general.md](general.md) | all | World history encyclopedias, chronologies |
| [eras.md](eras.md) | `era` | Period boundaries, regional periodizations |
| [civilizations.md](civilizations.md) | `civilization` | Founding/end dates, traits, scripts, regions |
| [events.md](events.md) | `event` | Event dates, categories, civilizations involved |
| [figures.md](figures.md) | `figure` | Birth/death dates, domains, civilizations |
| [campaigns.md](campaigns.md) | `campaign` | Campaign dates, battles, commanders, outcomes |
| [sites.md](sites.md) | `site` | Occupation dates, discovery dates, site types |
| [trade-routes.md](trade-routes.md) | `trade` | Route periods, commodities, endpoints |
| [calendars.md](calendars.md) | `calendar` | Calendar types, epochs, leap rules |
